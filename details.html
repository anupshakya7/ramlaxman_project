Web Design & Development

The process of building a website is referred to as web design and development under this general heading. It requires the two key skill sets of web design and web development, as the name would imply. Web design determines the look and feel of a website, while web development determines how it functions.

To store and manage all the data that is received through the front end of a website, a back end is required. Therefore, whether a user makes a purchase or completes a form on the front end of the website, they are entering information into an application. And that information is stored in a database that lives on a server.
